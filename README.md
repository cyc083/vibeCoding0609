# 台灣常識嘴巴快問快答

使用 p5.js、ml5.js FaceMesh 和 webcam 製作的 10 題是非題小遊戲。

## 操作方式

1. 用 VS Code 開啟這個資料夾。
2. 在終端機執行：

   ```bash
   python3 -m http.server 8000
   ```

3. 用瀏覽器開啟 `http://localhost:8000`。
4. 允許 webcam 權限。
5. **張開嘴**代表 `YES`，**閉上嘴**代表 `NO`。

## 遊戲規則

- 題目主題是台灣常識，共 10 題。
- 每一題都是是非題，畫面會提示「請作答」。
- 答對會顯示 3 秒特效，再進入下一題。
- 答錯不換題，請再做一次嘴巴動作。
- 遊戲會在 webcam 畫面上繪製臉部輪廓與嘴巴骨架節點。

## 注意

webcam 通常需要在 `localhost` 或 HTTPS 環境才可使用。若直接雙擊 `index.html` 用 `file://` 開啟，瀏覽器可能會擋下相機權限。
